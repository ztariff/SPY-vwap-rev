SUPPORT & TROUBLESHOOTING
==========================

Issue: "No data available" on charts
  Cause: Chart data fetch from Polygon API failed
  Fix: Check internet connection, verify API key is valid

Issue: Calendar appears empty
  Cause: Active filter may be hiding all trades
  Fix: Click "All Trades" filter button to reset

Issue: Page loads slowly
  Cause: Large dataset rendering all at once
  Fix: Use strategy cards or filter buttons to narrow view

Issue: Charts not updating
  Cause: API rate limiting or connectivity
  Fix: Wait 60 seconds for cache to expire, then retry
