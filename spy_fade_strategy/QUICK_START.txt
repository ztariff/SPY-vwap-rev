    • Entry/exit markers
    • Entry threshold, target, stop levels
    • Full trade metadata (prices, times, VIX, etc.)

STRATEGY SUMMARY:
  Table at bottom showing each strategy's:
  - Number of trades
  - Win rate
  - Average P&L
  - Total P&L %
  - Sharpe ratio
  - Profit factor

Click any filter button to isolate trades:

Direction:
  [Buy Dip] — Show only long trades (below VWAP)
  [Fade] — Show only short trades (above VWAP)

  [0.3x], [0.4x], [0.5x], [0.8x] — Entry sensitivity

Outcome:
  [Winners] — Only profitable trades
  [Losers] — Only losing trades

All filters update in real-time. Click another filter to switch.
Click [All] to reset.

INTERPRETING THE INTRADAY CHART:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Candlesticks:
  Green = price close > open (bullish bar)
  Red = price close < open (bearish bar)

Lines:
  Orange = VWAP (volume-weighted average price)
  Blue dashed = Entry threshold (VWAP ± ATR*multiplier)
  Green dotted = Target level (exit goal)
  Red dotted = Stop level (risk limit)

Markers:
  Blue circle = Trade entry signal + price
  Green circle = Profitable exit
  Red circle = Loss exit

Volume:
  Below chart = green/red bars showing trading volume

Hover crosshair:
  Move mouse over chart to see exact OHLC values at any time

To rebuild dashboard with updated trade data:
  1. Update stock_frontside_trades.json with new trades
  2. Run: python3 build_stock_dashboard.py
  3. New stock_dashboard.html will be generated

To modify colors:
  Edit CSS in build_stock_dashboard.py (look for #0f1117, #26a69a, #ef5350)
  or directly in stock_dashboard.html <style> section

To add new filters:
  Edit STRAT_LABELS in build_stock_dashboard.py
  Add new filter buttons in <div class="filter-group">
  Implement filter logic in getFiltered() function
